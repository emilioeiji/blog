    O style que utilizei foi o Nasa e o seu download pode ser feito aqui. Para efetuar a instalação do style é simples, basta descompactar o arquivo no diretório ~/.fluxbox/styles

    Papel de Parede

    Esse aqui é o papel de parede que estou usando, achei em uma busca rápida pelo google.

    Para efetuar a troca do papel de parede basta usar o seguinte comando fbsetbg /caminho/do/arquivo.jpg

    No meu debian não precisei alterar nenhuma configuração no style para salvar o papel de parede, mas se estiverem com problema podem seguir essa dica aqui.

    Tema para o conky

Copie e cole todo o conteúdo para seu arquivo conkyrc
Para colocar o conky na inicialização, edite o arquivo
~/.fluxbox/startup acima da linha *exec fluxbox *\ o comando:

    conky -c ~/.conky/conkyrc &

    Lembre-se de indicar o arquivo no local corretamente para que o conky busque as configurações corretamente.
