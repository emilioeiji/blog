    100% Linux Parte 2

    Como postei aqui, estou na batalha para me converter 100% em Linux e uma das minhas dificuldades que consegui superar hoje, na verdade se eu tivesse dado uma pesquisadinha mais a fundo teria conseguindo solucionar antes, mas eu confesso que estava procrastinando] era o uso do do M$ Silverlight, como é um plugin desenvolvido para rodar nos navegadores no windows já era de se esperar que não teria suporte no Linux.

    A minha primeira tentativa em conseguir rodar o silverlight netflix-desktop, já que o mesmo usa o silverlight para exibir os seus vídeos, mas sem sucesso, não sei se é alguma coisa com o 64 bits mas após efetuar a instalação do pacote ele simplesmente não abre, tentei instalar na versão 32 bits que tenho no virtualbox e funcionou, mas como remover a versão 64 e colocar a 32 está fora de questão essa solução não resolveu o meu problema, mas pra quem use o sistema 32 bits ou em alguma outra distribuição possa funcionar.

    sudo apt-add-repository ppa:ehoover/compholio
sudo apt-get update
sudo apt-get install netflix-desktop

    Pesquisei um pouco mais e encontrei o plugin para o FireFox chamado Novell Moonlight, que foi postado na primeira parte, até chegou até a carregar a página que eu preciso mas na hora de executar o vídeo deu erro de codec, até consegui encontrar alguma referencia no site da Microsoft desse codec para linux mas na hora de efetuar o download retornava que o codec não estava mais disponível. Também abandonei essa tentativa.

    Parei e pensei comigo, vou emular o firefox no wine e instalar o silverlight nele :) [STRIKEOUT:como não tinha pensado nisso antes????] mas como nada é do jeito que queremos :p lógico que não deu certo... Até a instalação do firefox foi tranquilo, o problema começou na hora de instalar o bendito silverlight :/ pesquisei em vários foruns e blogs mas nenhuma dica fez o bendito funcionar e novamente abandonei essa tentativa.

    Indo um pouco mais a fundo encontrei o pipelight :) "Um projeto que traz o Silverlight para linux em qualquer navegador que suporte a API Netscape Plugin, nele você consegue rodar qualquer serviço que requer o silverlight".

    sudo apt-add-repository ppa:pipelight/stable
sudo apt-get update
sudo apt-get install --install-recommends pipelight-multi
sudo pipelight-plugin --update

    Para ativar o plugin, execute o seguinte comando:

    pipelight-plugin --enable silverlight

    Se por algum motivo você desejar desabilitar o plugin, execute o seguinte comando:

    pipelight-plugin --disable

    Por padrão o plugin executa a versão 5.1 do silverlight, mas se desejar é possível rodar em uma versão anterior por exemplo:

    sudo pipelight-plugin --disable silverlight --enable silverlight5.0

    A baixo vou listar como habilitar algumas extensões para o pipelight, o nome já diz +- do que se trata:

    pipelight-plugin --enable flash
pipelight-plugin --enable widevine
pipelight-plugin --enable viewright-caiway
pipelight-plugin --enable vizzedrgr
pipelight-plugin --enable unity3d

    Temos a opção também de efetuar uma checagem no sistema para saber se o pipelight irá fuir:

    pipelight-plugin --system-check

    Constantemente o pipelight libera novos plugins e para conseguir ajuda rode o seguinte comando:

    pipelight-plugin --help

    E para habilitar os plugins:

    pipelight-plugin --enable nome\_do\_plugin

    Alguns plugins precisam ser desbloqueados antes de serem habilitados e para isso rode:

    sudo pipelight-plugin --unlock nome\_do\_plugin

    Seguindo os passos acima para instalar e habilitar o plugin basta fechar e abrir o navegador que tudo funcionará perfeitamente.
