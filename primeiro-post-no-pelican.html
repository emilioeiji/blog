    Primeiro post no pelican

    sáb 07 março 2015 pelican /

    Bom galera, estou tentando deixar o blog mais hardcore e estou migrando do wordpress para pelican. Não que o WP seja ruim ou não estivesse atendendo as minhas necessidades, mas como estou engajado em aprender python nada melhor doque fazer tudo em python.

    Como base nessa empreitada me inspirei no blog de um dos criadores do Hack N' Cast, Magnun Leno e segui algumas dicas postadas por ele aqui.

    O processo é bem trabalhoso pois temos que parametrizar vários detalhes, mas nada que com algumas horas de sono a menos não consigamos resolver. Consegui rodar o make html do pelican sem erros, mas ainda tenho que corrigir vários links e referencias, artigo por artigo. Mas que eu estou querendo testar em primeiro momento é a mensão aos códigos, ainda não descobri como fazer para destacar o código, coisa de noob que está começando a trabalhar com a ferramenta agora :p

    Bom abaixo vai um teste:

    .. code:: python

        print('Teste de sintax python no pelican.')

    print('Teste de sintax python no pelican.')

    .. code-block:: python

   print('Teste de sintax python no pelican.')

    print('Teste de sintax python no pelican.')

    #!python
print('Teste de sintax python no pelican.')

    #!python print('Teste de sintax python no pelican.')

    Como nenhuma das opções acima não deu certo, acredito que não seja problema problema ao escrever o artigo, visto que o código está sendo destacado apenas não está rolando o syntax highlight.

    Bom, mas vou continuar testando até conseguir :p Agora já são 18:51 hora de começar a se arrumar pra ir na missa, vamos ver que horas vou conseguir resolver essa problema hehehehehe.

    Bom, voltei agora 23:50, após assistir a missa e comer muuuuito...

    Agora o jeito é tomar um café espresso pra dar uma despertada e focar na pesquisa pra descobrir como funciona o syntax highlight desse trem.

    <print('Teste de sintax python no pelican.')>

    print "Teste de sintax python no pelican."

    print('Teste de sintax python no pelican.')

    Depois de muito quebrar a cabeça eu finalmente descobri qual era o problema com o syntax highlight do pelican, o Tema. Pois é... o tema que mais gostei não faz o destaque dos código :( O tema que estava utilizando era o BT3, estou usando provisóriamente o blueidea até encontrar outro similar ao BT3.
