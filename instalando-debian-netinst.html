    Instalando Debian Netinst

    Dom 29 março 2015 linux / debian / 100-linux / netinst / instalação / instalando /

    Boas galera, como estou precisando instalar o Debian para estudos, abaixo vou mostrar como fazer a instalação do netinst do zero.

    Para efetuar o download da iso clique aqui e após o término do download podemos gravar a iso em um cd ou criaros um pendrive de boot .

    Em seguida vamos dar boot na maquina e teremos a tela abaixo:

    Temos as seguintes opções de instalação:

    1. Install - Iniciará o mode de instalação padrão em modo texto.
    2. Graphical install - Iniciará a instalação em modo gráfico, um pouco mais amigável mas nada de diferente do modo padrão.
    3. Advanced options
      1. Modo Expert - Iniciará a instalaçao no modo very hard, onde se tem controle de todos os os processos que serão executados durante a instalação, indicado para usuários avançados.
      2. Graphical Expert - Iniciára a instalação no modo hard com uma interface gráfica.
    4. Help - Obtem ajuda do sistema na instalação.
    5. Install with speech synthesis - Inicia no modo de acessibilidade, onde um assistente "lê" oque está aparecendo na tela.

    Vou efetuar a instalação pelo modo padrão, selecionando Install, após isso ira carregar as telas onde selecionaremos a opção de linguagem do sistema, localidade e teclado.

    Irá carregar alguns módulos do sistema, configurar o ip e em seguita irá para a tela onde definiremos o nome do computador e o domínio.

    Após definirmos o nome vamos para a senha do root, cadatro do nome de um usuário e senha para o mesmo.

    Agora vamos definir o fuso horário.

    Na criação das partições devemos ter uma certa atenção caso tenhamos outros SO's instalados, visto que se selecionarmos a opção Assistido - usar disco inteiro todos os dados do disco serão apagados e dá o maior trabalho recuperar os arquivos :p

    Nesta etapa temos as seguintes opções:

    1. Assistido - usar o disco inteiro - Essa opção serve para usuários iniciantes que não tem experiencia com particionamento de discos ou igual estou fazendo agora apenas para testes. :)
    2. Assistido - usar o disco inteiro e configurar LVM - Essa opção serve para configurarmos o LVM (Gerenciamento de Volume Lógico) que facilida em situações que após instalar o sistemas precise alterar o tamanho do volume.
    3. Assistido - usar disco todo e LVM criptografado - Faz a junção das duas primeiras opções e faz a criptografia dos volumes.
    4. Manual - Aqui efetuamos as configurações manualmente das partições e volumes, quando instalamos em servidores ou desktops e precisamos definir os tamanhos das partições e o ponto de montagem para cada partição.

    Após o procedimento das partições é efetuado a instalação do sistema básico e em seguida selecionamos em qual repositório queremos buscar os nossos pacotes para que seja configurado o apt.

    Em seguida é solicitado se quer participar do consurso de utilização de pacote, que ajuda no desenvolvimento do sistema oferecendo informações anonimas.

    Após isso vamos para a tela de seleção de software, onde vamos selecionar oque queremos instalar, no meu caso como é uma maquina para teste não quero instalar nada além do sistema básico, portanto vou deixar marcado apenas a opção de Utilitários standard de sistema.

    Caso queria instalar um desktop completo deve marcar a opção Debian desktop environment e caso a instalação esteja ocorrendo de um laptop marcar a opção Laptop

    Temos outras opções para instalarmos como web server, servidor de impressão, sql db, file server, mail server, ssh server. Eu particularmente gosto de instalar tudo depois que finalizar a instalação do sistema.

    Agora vamos somos questionados se queremos instalar o gerenciador de boot GRUB.

    E pronto, acabamos de finalizar a instalação do Debian Netinst, muito fácil não?

    Agora já temos a nossa tão querida telinha preta.

    Após a instalação a primeira coisa que faço é deixar o bash coloridinho :p editando o bash /root/.bashrc descomentando as linhas:

    export LS_OPTIONS='--color=auto'
eval "`dircolors`"
alias ls='ls $LS_OPTIONS'
alias ll='ls $LS_OPTIONS -l'
alias l='ls $LS_OPTIONS -lA'

alias rm='rm -i'
alias cp='cp -i'
alias mv='mv -i'

    O segundo passo é instalar o vim para facilitar e edição dos arquivos.

    apt-get install vim

    e após a instalação descomento a linha syntax on no arquivo /etc/vim/vimrc para destacar a sintax nos arquivos de configurações.

    Agora vou procurar um ambiente gráfico para brincar e provavelmente irá tornar outro post do blog.
